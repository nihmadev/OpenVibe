# Security Policy

## Supported Versions

We currently provide security updates for the latest stable release.

| Version | Supported |
| ------- | --------- |
| 1.x     | ✅        |
| < 1.0   | ❌        |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please report it privately:

- **Email:** `lolz@nihmadev.fun`
- **Telegram:** `@nihmadev`

**Please do not report security vulnerabilities through public GitHub issues.**

We will acknowledge receipt within 48 hours and provide a timeline for a fix. You should expect to hear back from us within 5 business days.

## What to Include

- Type of vulnerability
- Steps to reproduce
- Affected versions
- Any potential impact

## Disclosure

We follow a 90-day disclosure window from the time a fix is released. This gives users time to update before details are made public.
