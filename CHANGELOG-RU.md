# История изменений (Changelog)

Все ключевые изменения проекта OpenVibe документируются в данном файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/),
и этот проект придерживается [Семантического версионирования](https://semver.org/lang/ru/).

## [1.3.0] — 15.07.2026

### Добавлено

- **Интеграция Model Context Protocol (MCP)**:
  - Полноценный клиентский бэкенд-движок MCP (`crates/mcp`) с поддержкой транспорта JSON-RPC и управления локальными stdio-серверами.
  - Специализированная панель настроек MCP (`McpSettingsPanel.tsx`) для добавления, редактирования, включения/выключения stdio-серверов, управления переменными окружения и аргументами запуска.
  - Индикатор статуса MCP в заголовке окна (Titlebar) с отображением активных серверов, доступных инструментов и текущего состояния подключения.
  - Визуализация выполнения инструментов в чате с ИИ-агентом (`AgentToolView.tsx`) со стримингом аргументов в реальном времени, выводом ответов, задержкой выполнения и статусной индикацией.

- **Быстрый поиск по коду (Search in Code)**:
  - Высокопроизводительная панель поиска по содержимому файлов (`SearchInCode.tsx`) с двумя режимами отображения: виртуализированный список (`@tanstack/react-virtual`) и древовидная структура папок.
  - Поддержка многопоточного обхода файлов (`jwalk`), учет правил `.gitignore`, кэширование результатов, поиск по регулярным выражениям (Regex), учет регистра, поиск целых слов и фильтрация по шаблонам Include/Exclude.
  - Подсветка найденных совпадений в коде и мгновенный переход на нужную строку файла прямо в редакторе Monaco Editor.
  - Поддержка горячих клавиш (`Ctrl+Shift+F`) и интеграция с палитрой команд (`Command Palette`).

- **Умная генерация контекста 2 (SCG2 — Smart Context Generation 2)**:
  - Встроенный движок интеллектуальной сборки контекста (`crates/scg2`) с легким AST-парсингом символов (структуры, функции, классы, интерфейсы, трейты) для языков Rust, TypeScript, JavaScript, Python и Go.
  - Многофакторная система оценки актуальности кода: затухание по времени обращений к файлам, фокус активных вкладок, близость курсора ввода, ошибки редактора (diagnostics) и неотправленные git-изменения (git delta).
  - Автоматический трекер активности редактора (`scg2Tracker.ts`), передающий события через Tauri IPC (`scg2_push_event`) для непрерывного формирования наиболее релевантных контекстных сниппетов в промпт агента.

- **Модуль рабочей области Git (`crates/git`)**:
  - Выделенный Rust-крайт для проверки состояния git-репозитория, определения имени ветки, отслеживания измененных файлов (staged/unstaged), расчета diff и анализа частоты редактирования файлов.

- **Кастомные правила проекта (.viberules)**:
  - Нативная загрузка и внедрение в системный промпт правил проекта из файлов `.viberules`, `AGENTS.md` и `.cursorrules` из корня открытой рабочей области.

- **Новые темы оформления Gruvbox**:
  - Добавлено 3 варианта цветовой темы Gruvbox: **Gruvbox Hard**, **Gruvbox Medium** и **Gruvbox Soft**.

- **Расширенные настройки кастомных AI-провайдеров**:
  - Гибкая настройка собственных провайдеров: возможность загрузки и выбора SVG-иконки, указание URL для получения списка моделей, установка кастомных JSON-заголовков авторизации и тонкая настройка параметров сэмплинга (temperature, top_p, penalty).

- **Оптимизация предзагрузки типов в редакторе**:
  - Вынос логики предзагрузки типов TypeScript в отдельный крайт `crates/editor`, благодаря чему загрузка файл-деклараций `.d.ts` происходит за один пакетный запрос по IPC.

### Исправлено и улучшено

- Исправлен критический баг с бесконечным циклом в подсветке совпадений поиска (`syntaxHighlightLine`) при пустом поисковом запросе.
- Настроены правила ESLint для React Hooks и автоформатирование Prettier для стабильного прохождения CI-сборки.
- Исправлены все предупреждения и строгости Rust Clippy (`unnecessary_sort_by`, `manual_ok_err`, `needless_borrow`, `manual_strip`, `collapsible_if`, `map_flatten`) во всех 11 крайтах проекта.
- Добавлена конфигурация `clippy.toml` с порогом `too-many-arguments-threshold = 12`.
- Рендеринг SVG-иконок переведен на единую модульную библиотеку компонентов (`src/components/icons/`).
- Полностью обновлены переводы и документация для функций MCP и поиска по коду на 38 поддерживаемых языках интерфейса.

## [1.2.0] — 11.07.2026

### Добавлено

- Первоначальный публичный релиз OpenVibe
- AI-агент с поддержкой чтения/записи/редактирования файлов, выполнения команд в терминале и поиска по коду
- Редактор Monaco Editor с поддержкой вкладок
- Настоящий PTY-терминал на базе xterm.js
- Сохранение истории чатов и проектов на базе SQLite
- 38 языков интерфейса, 17 цветовых тем, 230 иконок типов файлов
- 11 встроенных AI-провайдеров + поддержка любых OpenAI-совместимых эндпоинтов
- Векторный поиск по коду (fastembed) и поиск по регулярным выражениям
- Drag-and-drop файлов, прикрепление файлов через @-упоминания
- Настраиваемые горячие клавиши, масштабирование интерфейса, звуки завершения задач
- Кроссплатформенные сборки (Windows, macOS, Linux)
- Автоматизированный CI/CD пайплайн на GitHub Actions
